Horizontal and vertical spacing

Control horizontal and vertical spacing of an element using the mx-{size} utilities.

              <div class="py-20"></div>

Responsive

To control the margin of an element at a specific breakpoint, add a {screen}: prefix to any existing margin utility. For example, adding the class md:mt-20 to an element would apply the mt-20 utility at medium screen sizes and below.

            
              <div class="mt-30 md:mt-20 sm:mt-10"></div>

Reveal Animations

Single Animation

Template comes with several build in section reveal animations. To add or change an element’s reveal animation, you need to change [data-anim] attribute. 

            
              <div data-anim="slide-up"></div>
              <div data-anim="slide-down"></div>
              <div data-anim="slide-left"></div>
              <div data-anim="slide-right"></div>
              <div data-anim="fade"></div>
              <div data-anim="img-right"></div>

If animation doesn't work properly try to wrap the element you want to animate.

            
              <div data-anim="slide-up">
                Element You want to animate
              </div>

Animation Container

If you want to animate multiple elemets at once, wrap them in data-anim-wrap container and use data-anim-child insted of data-anim attribute. 

            
              <div data-anim-wrap>
                <div data-anim-child="slide-up"></div>
                <div data-anim-child="slide-up"></div>
                <div data-anim-child="slide-up"></div>
              </div>

Delay

By adding additional delay- property, you can change animation delay. Every next value adds 100 milliseconds. 

            
              <div data-anim="slide-up delay-1"></div>
              <div data-anim="slide-up delay-2"></div>
              <div data-anim="slide-up delay-3"></div>
              ...
              <div data-anim="slide-up delay-20"></div>

              <div data-anim-child="slide-up delay-1"></div>
              <div data-anim-child="slide-up delay-2"></div>
              <div data-anim-child="slide-up delay-3"></div>
              ...
              <div data-anim-child="slide-up delay-20"></div>

Parallax Effects

Use [data-parallax] attribute with value on parallax container and [data-parallax-target] to specify parallax item. 

          
            <div data-parallax="0.7" class="h-lg">
              <div
                data-parallax-target
                class="bg-image js-lazy"
                data-bg="some.jpg"
              ></div>
            </div>

Sass Source Files

This requires basic technical skills.

Boxcar comes with sass source files and simple gulp setup for it. To be able to edit sass files and compile them in css follow this steps:

1. Make sure that you have node.js installed, node.js.

2. Open terminal in the template folder.

3. Run npm install.

4. Once the installation is ready, run gulp build. This will compile sass into css files and activate local server for better development process. Now you can make changes in scss files.

After that you can use gulp server command instead of gulp build6 to run the local server for development.

If you have any questions or concerns, you can submit a support request, we usually respond very quickly :)
